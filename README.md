# OIBSIP PROJECT 
# Simple Calculator

This project is a **basic web calculator** built using **HTML, CSS, and JavaScript**. It provides an interactive interface to perform simple arithmetic operations including addition, subtraction, multiplication, and division.

## ✨ Features
- Clean and responsive UI
- Real-time input display
- Supports decimals and backspace
- Clear All and Evaluate functions

## 📂 Files
- `CALCULATOR - Copy.html`: The main calculator web page

## 🚀 How to Use
You can access the calculator directly using GitHub Pages (link below) or by opening the HTML file locally in your browser.
